### Install
```npm install```

### Run
* ```npm run dev```
* Go to `localhost:8080` in the web browser
 
### Select/Edit :, 
1. Click On Mesh *Cube*, *Cylinder*, *IcoSphere* to Pick or Select From Scene in Heirarchy Panel
2. Adjust mesh transform and properties in inspector


### Bouncing Animation
1. Animation will play on any selected node in heirarchy 
2. Adjust animation properties and click on play
3. Before playing node bounce animation I have done reset camera view via animation
4. On animation complete click on Back button to reset
